# CFG GENERATOR CSGO

## Descrição

A CS:GO cfg generator

## Como Usar

### Instalando

Instalar Requisitos:
```pip install requirements.txt```

## Abrindo

Abrir o Cmd ou PowerShell Na pasta e rodar o comando: ```python main.py```
